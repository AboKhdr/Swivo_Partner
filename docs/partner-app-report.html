تبويب البيت Dashboard

الصفحة الرئيسية وملخّص العمليات اليومية لحظياً.

🏠

1. لوحة التحكم DashboardScreen

الوظيفة: الصفحة الرئيسية وملخّص العمليات لحظياً.

تعرض:
  • بطاقة إيراد اليوم بالريال (﷼).
  • إحصائيات: طلبات معلّقة، نشطة، إجمالي اليوم.
  • قائمة الطلبات المعلّقة + البايكرز على الخدمة.
4 أزرار وصول سريع اتصال بالبايكر سحب للتحديث
🔔

2. الإشعارات NotificationsScreen

الوظيفة: صندوق وارد لكل الإشعارات.

تعرض:
  • مؤشّر "غير مقروء" + شارة نوع ملوّنة.
  • الوقت ("منذ X دقيقة") + العنوان والنص.
انتقال برابط عميق تعليم الكل كمقروء
③

تبويب الطلبات Orders

قلب التطبيق: استقبال الطلبات وتوكيلها ومتابعتها حتى الإكمال.

📋

3. قائمة الطلبات OrdersScreen

قائمة قابلة للبحث (بالاسم/رقم الطلب) والفلترة بالنوع والحالة.

لكل بطاقة: رقم الطلب (قابل للنسخ)، شارة حالة ملوّنة، الخدمة، الموقع، الوقت، المبلغ.

تمرير لا نهائي (20/صفحة)فلتر النوع: موبايل / في الموقع
🧾

4. تفاصيل الطلب OrderDetailsScreen

عرض كامل لطلب واحد مع أزرار تتغيّر حسب الحالة والنوع.

بطاقات: الخدمة، العميل، السيارة واللوحة، الخريطة (موبايل)، صور قبل/بعد (الموقع)، الدفع، الخط الزمني.

الحالةالنوعالإجراء
PENDING_PARTNERالكلرفض + قبول وتوكيل
ACCEPTEDموبايلتوكيل بايكر
ACCEPTEDفي الموقعالتقاط صورة البداية
STARTEDفي الموقعالتقاط صورة النهاية
🔴

5. الطلب الوارد IncomingOrderScreen

نافذة عاجلة بكامل الشاشة عند وصول طلب جديد، مع عدّاد تنازلي.

تعرض: شارة "مباشر" نابضة، بيانات العميل/الخدمة/السيارة/السعر.

اقبل الطلبرفضرفض تلقائي عند انتهاء الوقت
🛵

6. توكيل بايكر AssignBikerScreen

نافذة سفلية لاختيار بايكر متاح لطلب الموبايل.

فرز: الأقرب مسافة / الأعلى تقييماً. لكل بايكر: الاسم، الطلبات النشطة، التقييم.

قاعدة: يظهر فقط البايكرز على الخدمة بلا طلبات نشطة.
✋

7. نافذة رفض الطلب RejectOrderModal

اختيار سبب الرفض (خيارات راديو) + حقل ملاحظة إلزامي عند اختيار "أخرى". زر التأكيد معطّل حتى اكتمال الشرط.
📷

8. مراجعة تخطّي الصور SkipReviewScreen

إدارة طلبات البايكرز بتخطّي تصوير قبل/بعد الغسيل.

شريط إحصائيات (معلّق/موافق/مرفوض) + تبويبات فلترة + بيانات البايكر والطلب والسبب.

موافقةرفض مع ملاحظة
④

تبويب العمليات Operations

مركز يربط 9 أقسام في 3 مجموعات، إضافةً لمفتاح القبول التلقائي للطلبات.

🗂️ مجموعة الإدارة

9. قائمة العمليات OperationsNavigator

المركز الذي يربط 9 أقسام + مفتاح القبول التلقائي (يُزامَن مع الخادم ويتراجع عند الفشل).

10. الخدمات Services

خدمات مجمّعة بفئات قابلة للطيّ، لكل خدمة 3 أسعار (صغيرة/متوسطة/كبيرة) + مدة + نوع (مغسل/جوال/كلاهما) + مفتاح تفعيل.

11. الباقات Packages

حزم خدمات ببانر واسم، اختيار خدمات وتحديد كمية كل خدمة مع ملخّص لحظي.

12. العروض Offers

عروض ترويجية مربوطة بخدمة، أسعار مخصّصة لكل حجم، تاريخ انتهاء اختياري، تفعيل/تعطيل، حذف فوري.

13. الإضافات Add-ons

خدمات بيع إضافي (Upsell) بسعر ثابت، أسماء عربي/إنجليزي، مربوطة بخدمة أساسية.

18. الفروع Branches

إدارة الفروع: بانر، عنوان، فرع أساسي، أيام وفترات العمل، تفعيل الخدمات، حذف نهائي. تنبيه إغلاق ±10د حول الصلاة.

🧑‍🔧 مجموعة الطاقم

15. الطاقم Staff

تبويبان: البايكرز والمديرون. حالة كل عضو + مفتاح On/Off Duty + إجراءات (إيقاف/تعطيل/حذف/إعادة تفعيل).

16. البايكرز + التفاصيل Bikers

قائمة البايكرز، والضغط ينقل لشاشة تفاصيل بها إحصائيات (نشطة/مكتملة/تقييم) وسجلّ طلبات مرقّم.

17. إضافة بايكر AddBiker

تسجيل/تعديل عضو: صورة، اسم وهاتف، الفرع التابع، وساعات العمل (الجمعة معطّلة افتراضياً للصلاة).

💳 المجموعة المالية

14. الاشتراكات Subscriptions

سجلّ اشتراكات العملاء (قراءة فقط): العميل، الباقة، السيارة، شريط استخدام (3/10)، تاريخ الانتهاء، الحالة.

19. التقييمات Reviews

المتوسط + توزيع 1–5 بمخطّط أعمدة، فلترة بالفرع، وبطاقات تقييم العملاء مع التعليقات.

20. المدفوعات Payments

3 بطاقات (الوارد/الصادر/الصافي)، فلتر زمني، ومعاملات مجمّعة بالتاريخ مع شارة "قيد التحويل".

⑤

تبويب الملف الشخصي Profile

إعدادات الحساب والتفضيلات والدعم.

21. الملف الشخصي

قائمة الإعدادات: الحساب، الإشعارات، المظهر، اللغة، الدعم، الشروط، تسجيل الخروج، حذف الحساب.

22. معلومات الحساب

تعديل الاسم الأول والأخير فقط (الهاتف والبريد للقراءة) + عرض رمز FCM مع زر نسخ.

23. الدعم

بطاقتا اتصال (هاتف وبريد) + نموذج إرسال رسالة (موضوع + رسالة إلزاميان).

24. الشروط والأحكام

أقسام قابلة للطيّ تُجلب من الخادم، مع نصوص احتياطية محلّية عند الفشل.
