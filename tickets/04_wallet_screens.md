# 04 — Wallet Screens

## الشاشات المشمولة
1. `WalletScreen` — عرض الرصيد + ملخص الأرباح + آخر المعاملات
2. `TransactionsScreen` — قائمة كاملة مع فلتر

│  🟢  ربح طلب #ORD-1234      + ﷼ 45.00 │  ← أخضر للإضافة
│      27 أبريل 2026، 11:00 ص            │

أنواع المعاملات وأيقوناتها:
| النوع | الأيقونة | اللون |
|---|---|---|
| order_earning | 💰 | أخضر |
| payout | 💸 | أحمر |
| commission | 📊 | برتقالي |
| refund | 🔄 | أزرق |

## API Endpoints — WalletScreen

### 1. جلب بيانات المحفظة والرصيد
```
GET /api/biker/wallet

Response:
{
  "success": true,
  "data": {
    "wallet": {
      "_id": "...",
      "balance": 1250.00,
      "currency": "SAR"
    },
    "transactions": [
      {
        "_id": "...",
        "type": "order_earning",
        "entryType": "credit",
        "amount": 45.00,
        "status": "completed",
        "reference": "ORD-1234",
        "description": "ربح طلب #ORD-1234",
        "createdAt": "2026-04-27T11:00:00.000Z"
      },
      ...
    ]
  }
}
```
> ملاحظة: `/api/biker/wallet` يعيد آخر 50 معاملة.

### 2. إحصائيات الأرباح
```
GET /api/biker/balance

Response:
{
  "success": true,
  "data": {
    "balance": 1250.00,
    "thisWeek": 320.00,
    "thisMonth": 850.00,
    "totalEarnings": 4200.00,
    "totalPayouts": 2950.00,
    "currency": "SAR"
  }
}
```

## 4.2 TransactionsScreen

### الوصف
قائمة كاملة بالمعاملات مع فلتر بالتاريخ والنوع.

### التصميم
```
┌──────────────────────────────────────┐
│  ← Header: "المعاملات"              │
│                                      │
│  ── Filters ────────────────────    │
│  ┌────────────┐  ┌───────────────┐  │
│  │ 📅 التاريخ │  │ النوع: الكل ▾ │  │  ← DateRangePicker + TypeFilter
│  └────────────┘  └───────────────┘  │
│                                      │
│  ── المعاملات ──────────────────    │
│                                      │
│  ── 27 أبريل 2026 ──────────────    │
│  ┌─────────────────────────────────┐ │
│  │ [TransactionCard]               │ │
│  └─────────────────────────────────┘ │
│                                      │
│  ── 25 أبريل 2026 ──────────────    │
│  ┌─────────────────────────────────┐ │
│  │ [TransactionCard]               │ │
│  └─────────────────────────────────┘ │
│  ┌─────────────────────────────────┐ │
│  │ [TransactionCard]               │ │
│  └─────────────────────────────────┘ │
│                                      │
│  [تحميل المزيد] ← عند الوصول لنهاية القائمة │
└──────────────────────────────────────┘
```

### TransactionCard (تفصيلي)
```
┌──────────────────────────────────────────┐
│  🟢 ربح طلب                              │
│  #ORD-1234  ─────────  + ﷼ 45.00        │
│  11:00 ص  ·  مكتمل ✓                    │
└──────────────────────────────────────────┘
```

### Filter Types (Dropdown)
```
الكل | أرباح | صرف | عمولة | استرداد
```

### API — TransactionsScreen
```
GET /api/biker/transactions?page=1&limit=20&startDate=2026-04-01&endDate=2026-04-27&type=order_earning

Query params:
- page: number (default: 1)
- limit: number (default: 20)
- startDate: ISO date string (optional)
- endDate: ISO date string (optional)
- type: 'order_earning' | 'payout' | 'commission' | 'refund' (optional)
- status: 'completed' | 'pending' | 'failed' (optional)

Response:
{
  "success": true,
  "data": [
    {
      "_id": "...",
      "type": "order_earning",
      "entryType": "credit",
      "amount": 45.00,
      "status": "completed",
      "reference": "ORD-1234",
      "description": "ربح طلب #ORD-1234",
      "createdAt": "2026-04-27T11:00:00.000Z"
    }
  ],
  "total": 48,
  "page": 1,
  "pages": 3,
  "hasNext": true,
  "hasPrev": false
}
```

## ملاحظات التصميم

- Balance Card: استخدم LinearGradient (primary → primaryDark) خلفية بيضاء للأرقام
- Infinite Scroll: عند الوصول لـ 80% من نهاية القائمة → جلب الصفحة التالية
- Empty State: إذا لا توجد معاملات → صورة + نص "لا توجد معاملات بعد"
- Pull to Refresh: لإعادة جلب البيانات
